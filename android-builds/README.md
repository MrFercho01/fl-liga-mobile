# Android builds

Guarda aqui temporalmente los APK o AAB generados para Android.

Ejemplos recomendados:
- fl-liga-preview.apk
- fl-liga-production.apk
- fl-liga-production.aab

Si vas a publicar la descarga dentro de la app, sube uno de estos archivos a una URL publica y configura:

EXPO_PUBLIC_ANDROID_APK_URL=https://tu-dominio.com/android/fl-liga.apk
